## 🌍 Deploy on GitHub Pages

1. Push the project to GitHub.
2. Open Repository Settings.
3. Navigate to Pages.
4. Select:

   * Source → Deploy from a branch
   * Branch → main
   * Folder → / (root)
5. Save changes.

Your site will be available at:

```text
https://YOUR_USERNAME.github.io/portfolio-website/
```
